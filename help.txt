~ Commands ~

Text Formatting

    { Bold - Bolds the message that you send }
    { Italic - Italicizes the message that you send }
    { Underline - Underlines the message that you send }

Utility

    { Avatar / Av / Pfp - Shows the avatar of a specified user }
    { Serverpfp - Shows the avatar of your current guild }
    { Bypass - Bypasses a linkvertise link (currently broken) }
    { Ip / Geoip / Iplookup - Sends information about an input ip address }
    { Dm - Dms a specified user your message }

Nuke
